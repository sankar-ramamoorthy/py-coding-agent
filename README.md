# py-coding-agent

A Python-based self-extending coding agent with Docker sandboxing, dynamic tool creation, and Ollama LLM integration.  

## Features

- CLI-driven coding agent
- Docker sandbox for safe execution
- Python `uv` package management
- Dynamic creation of Python tools
- Ollama LLM integration for natural language reasoning

## V1 Goals

- Run locally in a Docker container
- Read/write files and execute shell commands
- Install Python dependencies dynamically
- Create and use new Python tools

## Future (V2) Enhancements

- Multi-agent pods (planner, coder, tester)
- Tool registry and validation
- Tool memory index
- Automated testing of tools

## Getting Started

Clone the repo:

# powershell
git clone https://github.com/sankar-ramamoorthy/py-coding-agent.git
